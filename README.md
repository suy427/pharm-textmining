# 결과물

### 결과물 네이밍 규칙
#### 폴더 : [생성날자]크롤링한 기간 ([YYYYMMDDHHmmSS]from-to)
ex) [20190820223015]20160901-20161231

#### 텍스트 파일 : 크롤링한 기간.txt (from-to)
ex) 20160901-20161231.txt

#### skipped.txt : 성인인증 등의 이유로 직접적으로 크롤링 불가한 문서의 갯수

### 폴더 안 내용물
#### 1. 기간별 크롤링한 도큐먼트
#### 2. 성인인증 등으로 인해 크롤링하지 못한 도큐먼드 수
